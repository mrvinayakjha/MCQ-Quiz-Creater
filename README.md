## How to run :
### To run on base env
1. open terminal
2. pip install -r requirements.txt
3. create .env file and add GOOGLE_API_KEY="####### api key here #######"
4. python3 app.py

### To run on virtual env
1. conda create -n environment_name
2. conda activate environmant_name
3. and follow above steps
